## Overview

The `ConvexManager` lets you programmatically provision Convex projects, push schemas, manage environment variables, and deploy — all from code.

<Callout type="info">
  Requires a [Convex Team Access Token](https://dashboard.convex.dev) — get one from **Team Settings → Access Tokens**.
</Callout>

## Setup

```ts
import { ConvexManager } from 'tekimax-ts'

const convex = new ConvexManager({
  accessToken: process.env.CONVEX_ACCESS_TOKEN,
  teamId: process.env.CONVEX_TEAM_ID,  // optional — auto-resolved from token
})
```

## Create a Project

```ts
const project = await convex.createProject('my-ai-app', {
  deploymentType: 'prod',
  region: 'us-east',
})

console.log(project.deploymentUrl)
// => https://diligent-antelope-456.convex.cloud
```

## Set Environment Variables

```ts
await convex.setEnvVars(project.deploymentName, [
  { name: 'OPENAI_API_KEY', value: process.env.OPENAI_API_KEY! },
  { name: 'APP_URL', value: 'https://myapp.com' },
])
```

## Deploy Functions & Schema

```ts
const deployKey = await convex.createDeployKey(project.deploymentName)

convex.deploy(deployKey, {
  projectDir: './my-convex-app',
})
```

## Generate OpenAPI Spec

Generate a type-safe OpenAPI spec from your Convex deployment for use with Go, Java, or other language clients.

```ts
convex.generateOpenAPISpec({
  projectDir: './my-convex-app',
  output: 'convex-spec.yaml',
})
```

See [Convex OpenAPI docs](https://docs.convex.dev/client/open-api) for details on generating clients from the spec.

## List & Delete Projects

```ts
const projects = await convex.listProjects()

await convex.deleteProject(projects[0].id)
```

## API Reference

| Method | Description |
|--------|-------------|
| `createProject(name, opts?)` | Create a project with an initial deployment |
| `deleteProject(projectId)` | Delete a project by ID |
| `listProjects()` | List all team projects |
| `createDeployKey(deploymentName)` | Generate a deploy key for schema push |
| `setEnvVars(deploymentName, vars)` | Bulk set environment variables |
| `getEnvVars(deploymentName)` | List environment variables |
| `deploy(deployKey, opts?)` | Push functions & schema via CLI |
| `generateOpenAPISpec(opts?)` | Generate OpenAPI spec from deployment |
| `getTokenDetails()` | Resolve team info from access token |
| `getRegions()` | List available deployment regions |
