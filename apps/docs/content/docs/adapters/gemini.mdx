# Gemini Adapter

The `tekimax-gemini` package integrates with Google's Generative AI SDK (`@google/genai`).

## Installation

```bash
npm install tekimax-ts
```

## Usage

```typescript
import { Tekimax } from 'tekimax-ts';
import { GeminiProvider } from 'tekimax-ts';

const client = new Tekimax({
  provider: new GeminiProvider({
    apiKey: process.env.GOOGLE_API_KEY,
  })
});

const result = await client.text.chat.completions.create({
  model: 'gemini-1.5-pro',
  messages: [{ role: 'user', content: 'Hello Gemini' }]
});
```
