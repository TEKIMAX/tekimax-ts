# OpenAI Adapter

The `tekimax-openai` package enables support for OpenAI's official API.

## Installation

```bash
npm install tekimax-ts
```

## Usage

```typescript
import { Tekimax } from 'tekimax-ts';
import { OpenAIProvider } from 'tekimax-ts';

const client = new Tekimax({
  provider: new OpenAIProvider({
    apiKey: process.env.OPENAI_API_KEY,
    // organization: 'org-...' // Optional
  })
});

const result = await client.text.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello OpenAI' }]
});
```
